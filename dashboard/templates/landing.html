How It Works

SHA-256 Hashing

Passwords are never stored in plain text. Every password is hashed using the SHA-256 algorithm before being saved to the database.

Token Authentication

Upon login, a unique cryptographic token is generated and stored in your session, providing secure and stateless authentication.

Custom Middleware

A custom Django middleware intercepts every request to validate tokens and attach the authenticated user context automatically.
